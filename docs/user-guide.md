# WebClip Chrome 扩展 — 用户功能说明

## 功能

WebClip 是一个 Chrome 扩展，提供网页截图 + 批注绘制功能。

### 使用流程

1. **点击扩展按钮** — 浏览器工具栏上的 WebClip 图标
2. **拖拽选区** — 在页面上拖拽选择要截图的区域，未选区域被半透明遮罩覆盖
3. **绘制批注** — 在选区内使用各种绘图工具标注
4. **保存/复制** — 将选区截图 + 批注合成图片

### 绘图工具

| 工具 | 图标 | 说明 |
|------|------|------|
| 选择 | ☝ | 选中、移动、缩放已有批注 |
| 矩形 | ▭ | 绘制矩形框 |
| 圆形 | ◯ | 绘制椭圆 |
| 箭头 | → | 绘制带箭头的线 |
| 文字 | T | 输入文字批注（点击后直接输入） |
| 标签 | 💬 | 绘制带背景色的标签批注 |
| 画笔 | ✎ | 自由手绘 |
| 删除 | ✕ | 删除选中批注 |

### 样式设置

- **颜色**：7 种预设颜色（红/橙/黄/绿/蓝/紫/黑）
- **粗细**：3 种预设粗细（细/中/粗）
- **字号**：文字和标签工具专用，3 种预设字号

### 操作按钮

| 按钮 | 说明 |
|------|------|
| 保存 | 将选区截图 + 批注合成 PNG 下载，保存后自动关闭截图界面 |
| 复制 | 将选区截图 + 批注合成 PNG 复制到剪贴板，复制后自动关闭截图界面 |
| 重选 | 回到选区阶段，重新选择截图区域 |
| 关闭 | 关闭截图界面 |

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Z | 撤销 |
| Ctrl+Y / Ctrl+Shift+Z | 重做 |
| Delete / Backspace | 删除选中批注（编辑文字时不触发） |
| Escape | 取消选中 / 关闭选区 |

### 技术细节

- 选区最小 10×10 像素
- 批注画布使用 fabric.js v5
- Undo/Redo 采用快照式（canvas.toJSON/loadFromJSON），最大 100 步
- 工具栏位置：选区下方（空间不足时自动移到上方）
- 页面滚动在截图期间被锁定
- 关闭截图界面后，再次点击扩展按钮即可重新使用
